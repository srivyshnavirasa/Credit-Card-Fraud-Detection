#  Credit Card Fraud Detection using Machine Learning

##  Project Overview

This project uses Machine Learning to detect fraudulent credit card transactions. A Random Forest Classifier is trained on a highly imbalanced dataset to classify transactions as genuine or fraudulent. The project includes data preprocessing, exploratory data analysis, model training, and performance evaluation.

---

##  Objectives

- Detect fraudulent credit card transactions.
- Analyze transaction data.
- Build a Machine Learning model for fraud detection.
- Evaluate the model using different performance metrics.

---

##  Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
  
---

##  Dataset

- Credit Card Fraud Detection Dataset
- Total Transactions: **284,807**
- Features: **30**
- Target Variable:
  - **0** → Genuine Transaction
  - **1** → Fraudulent Transaction

---

## Project Workflow

1. Load the dataset.
2. Explore and preprocess the data.
3. Analyze fraud distribution.
4. Split the data into training and testing sets.
5. Train a Random Forest Classifier.
6. Evaluate the model using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
7. Predict fraudulent transactions.

---

## Model Performance

- Model: **Random Forest Classifier**
- Test Accuracy: **99.96%**
- Evaluation Metrics:
  - Accuracy
  - Precision
  - Recall
  - F1-Score
  - Confusion Matrix
